# Seller Studio — Карта промптов

## Workflow (как работает процесс)

1. **Пользователь называет фичу** → агент указывает папку на рабочем столе
2. **Пользователь открывает файл**, копирует промпт, тестирует в ChatGPT Image 2
3. **Пользователь присылает улучшенный промпт** → агент вставляет его в `api/_prompts.js` в блок `PROMPTS_RU`
4. **Агент деплоит** на VPS двумя командами

## Как вставить улучшенный промпт в систему

### Шаг 1 — Найти нужный ключ в файле
Открыть `api/_prompts.js` → Ctrl+F → найти ключ из таблицы (например `QUICK_CARD_PROMPT_NATURAL`)

### Шаг 2 — Заменить текст промпта
Заменить содержимое между backtick-ами `` ` `` ... `` ` `` на новый текст от пользователя.

### Шаг 3 — Обновить файл на рабочем столе
Скопировать новый текст промпта в соответствующий `.txt` файл в папке `Промты RUS\`.

### Шаг 4 — Задеплоить на VPS (только для `_prompts.js`)
```powershell
scp -i "C:\Users\LORD-KSON\Desktop\ТРЕКОПЁС\sanya_trekopes" "c:\Users\LORD-KSON\Kson Project\Все мои APPы\VTON-MVP\api\_prompts.js" root@186.246.29.31:/root/vton-mvp-app/api/_prompts.js
```
```powershell
ssh -i "C:\Users\LORD-KSON\Desktop\ТРЕКОПЁС\sanya_trekopes" root@186.246.29.31 "docker restart vton-mvp"
```

> ⚠️ `_prompts.js` **не требует docker build** — только `docker restart`. Это занимает ~10 секунд.
