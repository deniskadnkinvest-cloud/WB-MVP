## 🔥 ГЛАВНАЯ ТЕМА ЗАПРОСА — НОВЫЙ ФУНКЦИОНАЛ: Режим «Предметная Съёмка»

Это абсолютно новый режим приложения — **📦 Товары (Предметка)**. Раньше Seller Studio умел только одежду. Теперь — ВСЁ что продаётся на маркетплейсе.

### Что именно новое:

#### 1. Переключатель режимов
В шапке приложения появился переключатель:
- **👗 Одежда** — старый Virtual Try-On
- **📦 Товары (Предметка)** — НОВЫЙ режим предметной съёмки

Это означает: теперь одно приложение закрывает ВСЕ потребности селлера в фотографии.

#### 2. Одиннадцать категорий товаров
```
🧴 Косметика и уход     💊 БАДы и витамины     🕯️ Декор и свечи
📱 Электроника и чехлы  🐾 Зоотовары           🌹 Парфюмерия
📎 Канцелярия           💎 Ювелирные изделия   🍎 Продукты питания
🏋️ Спортивные товары    📋 Другое
```
**Особенность «Другое»:** если твой товар не попал ни в одну категорию — описываешь его сам текстом. Это открывает режим для ЛЮБОГО товара на маркетплейсе без исключений.

#### 3. Пять профессиональных композиций кадра
```
🖼️ Натюрморт (Спереди)  —  классический предметный кадр
📐 Flat Lay (Сверху)     —  вид сверху, модный формат для beauty/food
🔍 Макро (Крупно)        —  детальный кадр (текстура, этикетка, качество)
↗️ Диагональ (3/4)       —  динамичный ракурс
👉 Товар в руке          —  lifestyle-кадр, рука держит товар
```

#### 4. Шесть спецэффектов — это главная «вирусная» фича
```
❌ Без эффектов           —  чистый коммерческий кадр
💦 Брызги воды           —  для напитков, косметики, освежающих товаров
🧴 Мазок крема           —  для beauty-товаров, кремов, масок
🔥 Пламя и свечение      —  для свечей, духов, крепкого алкоголя
🌸 Лепестки цветов       —  для парфюмерии, натуральной косметики
💊 Капсулы рядом         —  для БАДов, витаминов, аптечных товаров
```
**Это убийственная фича**: фотограф не может создать «брызги воды» на фото без отдельной дорогой фотосессии. ИИ делает это за 30 секунд.

#### 5. Пять профессиональных сцен/фонов
```
✨ Clean Beauty        —  белый минимализм (Glossier/Fenty-стиль)
🌿 Organic Eco        —  натуральные материалы, бежевые тона
🏡 Скандинавский уют  —  тёплый домашний интерьер
🌆 Urban Tech         —  тёмный технологичный фон
💼 Рабочий стол       —  нейтральная деловая сцена
```

#### 6. Модель-человек + товар
Опциональная фича: можно добавить ИИ-модель человека в кадр. Примеры использования:
- Рука модели держит флакон духов
- Модель демонстрирует браслет на запястье
- Спортсмен с шейкером / бутылкой воды
- Девушка наносит крем на лицо

#### 7. Раскадровка для предметки (3 или 5 кадров)
Одним нажатием — сразу несколько ракурсов товара с разными композициями. Готовый мини-лукбук товара.

#### 8. Умное итеративное редактирование
После генерации можно текстом уточнить: «сделать фон темнее», «добавить блики», «убрать тени», «повернуть товар». Quick-теги: Убрать тени / Ярче свет / Темнее фон / Добавить блики / Добавить текстуру / Другой ракурс.

## 📋 Задание для тебя (Deep Think)

Включи режим глубокого обдумывания и выполни:

### 1. Анализ уникального позиционирования Product Mode

- Какой **ключевой момент «WOW»** в Product Mode, который невозможно показать в других продуктах? (подсказка: спецэффекты за 30 сек vs профессиональная студийная съёмка с реквизитом)
- Какой **эмоциональный крючок** для каждого сегмента: beauty-селлер / food-селлер / ювелир / tech-продавец
- Что является **«было-стало»** в предметке? (сухой белый фон от поставщика vs атмосферный lifestyle-кадр с эффектами)
- Почему **спецэффекты — это главная вирусная фича**, которой нет ни у кого на рынке?

### 2. Три ДЕТАЛИЗИРОВАННЫХ сценария видеоролика ТОЛЬКО про Product Mode

> **ВАЖНО:** Все три ролика посвящены ТОЛЬКО новому режиму предметной съёмки. Одежда/VTON не упоминается. Это отдельная рекламная кампания.

Каждый сценарий обязан содержать:

#### A. Метаданные ролика:
- Название и концепция в одном предложении
- Хронометраж (30 / 45 / 60 сек)
- Стиль съёмки (экранная запись / постановочный / motion graphics / микс)
- Платформа (Telegram / YouTube Shorts / Instagram Reels / VK Клипы)
- Настроение/вайб
- Музыка: жанр, темп, BPM, атмосфера (конкретно — «Lo-fi hip-hop 85 BPM», «Trap с нарастанием», «Phonk с ударами»)
- Целевой сегмент ЦА

#### B. Покадровая раскадровка (КАЖДЫЙ кадр описан подробно):

Для КАЖДОГО кадра:
- **[ЧЧ:СС — ЧЧ:СС]** | Тайминг
- **ВИЗУАЛ:** точное описание что видно на экране
- **ТЕКСТ НА ЭКРАНЕ:** точная формулировка, шрифт-настроение (жирный/лёгкий), цвет, позиция
- **ЗАКАДРОВЫЙ ГОЛОС / ЗВУКОВОЙ ЭФФЕКТ:** точный текст ЗКГ или описание звука
- **ПЕРЕХОД:** cut / zoom in / swipe / морф / fade / wipe

#### C. Психологическое обоснование:
- Почему именно эта структура работает?
- Какой психологический механизм вовлечения?
- Как ролик ведёт зрителя от «пролистываю» → «хочу попробовать»?

### 3. Главные визуальные моменты обязательные в роликах

Обязательно включи демонстрацию хотя бы одного из этих моментов в КАЖДЫЙ ролик:

- **«Эффект-момент»**: товар + выбор эффекта + МГНОВЕННЫЙ результат. Например: тюбик крема → выбираю «Мазок крема» → на экране появляется профессиональный кадр с мазком крема рядом.
- **«Категория-момент»**: прокликивание категорий → мгновенное переключение контекста (Косметика / Электроника / Ювелирные / Зоотовары)
- **«Было-стало»**: плохое фото товара от поставщика (светлый фон, плохой свет) → великолепный lifestyle-кадр из Seller Studio
- **«30 секунд»**: таймер на экране, который показывает реальное время генерации

### 4. Специфические идеи под конкретные сегменты

Придумай по одному **уникальному хуку** для каждого сегмента:
- **Beauty/Косметика**: как показать «Мазок крема» и «Брызги воды» максимально вирусно
- **Парфюмерия**: «Лепестки цветов» + атмосфера luxury за ₽50
- **Ювелирные**: как показать Macro + рука модели = ювелирный лукбук
- **Food/Продукты**: Flat Lay + натуральные цвета = аппетитный кадр
- **Зоотовары/Pet**: как показать что товар для животных выглядит premium

### 5. Рекомендации по продакшну и адаптации

- Что можно снять **самостоятельно** с телефоном?
- Что требует **минимального оборудования** (кольцевая лампа, держатель телефона)?
- Как **адаптировать один ролик** под разные платформы без пересъёмки?
- Какие **звуковые эффекты** максимально усилят момент генерации (WHOOSH, DING, REVEAL)?

## 🎯 Критерии идеального сценария

1. **Hook за 3 секунды** — зрителю НЕЛЬЗЯ дать пролистать. Лучший hook для предметки: показать плохое фото → мгновенно превращается в идеальное
2. **Показать, не рассказать** — максимум экранных демонстраций эффектов
3. **Понятно без звука** — 85% просмотров в Telegram и Instagram — без звука. Текст на экране несёт весь смысл
4. **Конкретика** — не «красивое фото», а «Брызги воды — без студии и реквизита»; не «быстро», а «30 секунд»
5. **Вирусная фича в центре** — спецэффекты = главная звезда ролика
6. **Язык продавца** — «карточка товара», «WB», «Ozon», «поставщик», «конверсия», не «нейронная сеть» и «multimodal»
7. **Один ролик — один WOW-момент** — не пытаться показать всё, но один момент должен взрывать мозг

## ⚠️ Чего НЕ делать

- НЕ смешивать с функционалом одежды/VTON — это отдельная кампания
- НЕ называть технические термины (Gemini API, React, Firebase, промпт)
- НЕ показывать скучный процесс ожидания — только таймлапс с таймером
- НЕ делать корпоративный скучный стиль (мы — не банк, не госсервис)
- НЕ длиннее 60 секунд — это контент для маркетплейс-сообществ в Telegram
- НЕ делать ролик только про одну категорию — Product Mode универсален
- НЕ обещать нереального — только реальные фичи, которые уже работают
