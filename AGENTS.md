# AGENTS.md — Инструкция для Codex / AI-агентов (VTON MVP)

Привет! Этот проект представляет собой интерактивную примерочную.

## Основная структура
- `src/` — React-фронтенд (интерфейс примерочной, загрузка фото).
- `server.js` — Express-сервер для обработки API запросов примерки.
- `server-autocatalog.js` — скрипт автоматической каталогизации товаров.
- `api/` — функции Inngest для асинхронного процессинга.
- `BUSINESS_BRIEF.md` — подробное описание бизнес-модели продукта и требований к MVP.

## Правила разработки
1. **Фоновые задачи**: Вся тяжелая логика (генерация, обработка изображений) вынесена в шаги Inngest (`api/inngest/`). Не делай долгих блокирующих вызовов в Express-эндпоинтах.
2. **Удаление фона**: `@imgly/background-removal` используется на стороне клиента для снижения нагрузки на сервер.
3. **Безопасность**: Все приватные ключи Firebase Admin SDK должны загружаться через переменные окружения, никаких жестко закодированных JSON-файлов ключей в коде.
