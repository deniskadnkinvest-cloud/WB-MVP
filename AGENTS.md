# AGENTS.md — Инструкция для Codex / AI-агентов (VTON MVP)

Привет! Этот проект представляет собой интерактивную примерочную на базе AI-генерации изображений.

## Основная структура
- `src/` — React-фронтенд (интерфейс примерочной, загрузка фото).
- `server.js` — Express-сервер для обработки API запросов.
- `api/` — API обработчики.
- `BUSINESS_BRIEF.md` — подробное описание бизнес-модели продукта.

---

## ⛔ ИНФРАСТРУКТУРА И ДЕПЛОЙ (VPS)

**ВНИМАНИЕ: ЭТОТ ПРОЕКТ БОЛЬШЕ НЕ НА VERCEL.**
Проект работает на **Timeweb Cloud VPS (Россия)** из-за требований ФЗ-406.
В качестве СУБД используется собственный **PostgreSQL**, а не Supabase.
Среда развертывания — Docker / PM2 / Nginx.

**КРИТИЧЕСКОЕ ПРАВИЛО:**
НИКОГДА не упоминай Vercel, Serverless Functions, Vercel Hobby plan, Supabase или Firebase в контексте этого проекта. Все ключи и переменные окружения хранятся в `.env` на VPS-сервере.

## ⚠️ КРИТИЧЕСКИЕ ПРАВИЛА (нельзя нарушать)

### Безопасность
- **НИКОГДА** не хардкодить API-ключи, токены, секреты в коде. Только `process.env.VAR_NAME`.

### Архитектура UID (критично для подписок)
- Все пользователи идентифицируются через **`tg_{id}`**, **`vk_{id}`**, **`ya_{id}`** — это стабильный UID.
- Subscription source: PostgreSQL table `subscriptions` joined to `users`.

### Платежная система (ЮKassa)
- Webhook endpoint: `api/payment-webhook-yookassa.js`
- IP-адреса ЮKassa в whitelist: `185.71.76.*`, `185.71.77.*`, `77.75.153.*`, `77.75.154.*`

### Лимиты квоты Gemini API
- Не вызывай Gemini API в циклах без явного `await` между итерациями.
- При ошибке `429` (quota exceeded) — возвращай понятное сообщение пользователю.
- Никогда не вызывай AI API в фоне без явного действия пользователя.

### generate-image.js
- Это самый большой и сложный файл проекта (~2900 строк). 
- **Любое редактирование** этого файла требует проверки: `npm run lint` после сохранения.

## Известные проблемы и их решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| `500 SyntaxError` на `/api/generate-image` | Незакрытый template literal в файле | `npm run lint`, найди строку с ошибкой, исправь |
| Подписка не активируется после оплаты | Webhook ЮKassa отклонён (неверный IP) | Проверь whitelist в `payment-webhook-yookassa.js` |
